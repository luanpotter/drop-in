The simplest "drop-in" replacement for the browser's default CSS.

Just [a few hundred lines of CSS](https://github.com/luanpotter/drop-in/blob/master/packages/drop-in-css/drop-in.css), all generically made to work with HTML structure as semantics, with minimal class applications for special cases.
